# Library Management System

A console-based library management system written in C. Allows users to add books, list all books, search by author and view total book count using a menu-driven interface.

## Features
- Add new books with name, author, pages and price
- List all books in the library
- Search and list books by author name
- Display total number of books
- Clean menu-driven interface

## What I Learned
- How to build a menu-driven program using do-while and switch-case
- How to use structs to store related data
- How to pass arrays and pointers to functions
- How to use strcmp for string comparison
- How to separate logic into multiple functions
